Download Docs
Download Fonts

Hundreds of fonts at your fingertips

With one single command, fonty helps you search, download, and install fonts from a variety of sources such as Google Fonts.

Manage Fonts

Manage fonts with the power of command line

Add or remove fonts from your system with the efficiency and extensibility of the command line.

Convert Fonts

Generate webfonts and stylesheets instantly

Convert fonts to the woff, woff2 formats and generate @font-face CSS rules to speedup your web development workflow.

Download

ALPHA
fonty can be installed using the command line with pip, the Python Package Manager.
pip install fonty
Requires Python 3. Mac and Windows only, Linux support is coming soon.
